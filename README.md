Documentation
-------------
`<leader>rn`: Rename symbol
`gd`: Go to definition
`gr`: Find references
`<leader>ca`: Perform code action (LSP suggestion)
`<leader>s`: Full text search on working directory
`<leader>sp`: Full text search currently active project (=git repo)
`<leader>sg`: Full text search globally (complete filesystem search)
`<leader>sc`: Search for commands / keymaps
`<leader>sb`: Search for buffers
`<leader>sf`: Search for files by name
